# 寰宇回响 - 沉浸式星象互动占卜装置

## 项目简介

基于"寰宇回响"项目功能描述制作的可交互HTML网页，展示星象占卜装置的完整功能演示。当代码邂逅星空，传统文化焕发数字生命力。

## 核心功能

### 🌌 3D星空操控
- 基于Three.js的3D虚拟星空场景
- 支持鼠标拖拽旋转、滚轮缩放
- 可点击星座查看详情信息
- 北斗七星、猎户座等星座展示

### 🔮 AI智能占卜
- 完整的占卜流程演示
- 模拟NASA数据库和紫金山天文台数据查询
- AI分析过程可视化
- 个性化星象解读生成

### 🎤 多模态交互
- 语音输入功能（需浏览器授权）
- 文字输入支持
- 手势操控模拟
- 音效和语音播报

### ✨ 沉浸式体验
- 动态星空背景粒子系统
- 流畅的页面切换动画
- 玻璃质感UI设计
- 响应式布局适配

## 技术实现

### 前端技术栈
- **HTML5**: 语义化结构
- **CSS3**: 现代样式、动画、渐变效果
- **JavaScript (ES6+)**: 模块化交互逻辑
- **Three.js**: 3D星空场景渲染
- **GSAP**: 高性能动画库
- **Web APIs**: 语音识别、语音合成、全屏等

### 核心组件
1. **CosmicResonanceApp**: 主应用控制器
2. **StarField3D**: 3D星空场景管理
3. **VoiceInputSimulator**: 语音输入模拟
4. **DivinationEngine**: 占卜流程控制
5. **TextToSpeechSimulator**: 语音播报

## 使用说明

### 启动方式
1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 使用Node.js
   npx serve .
   ```

### 操作流程
1. **欢迎页面**: 了解项目功能，点击"开始体验"
2. **星空操控**: 拖拽旋转星空，点击星座，启动占星术
3. **信息输入**: 输入生日和占卜问题，支持语音输入
4. **占卜过程**: 观看AI分析过程动画
5. **结果展示**: 查看星象解读、插画、签文，支持语音播报

### 快捷键
- **空格键**: 切换音效开关
- **ESC键**: 退出全屏模式
- **数字键1-3**: 快速导航到对应页面

## 注意事项

1. **音频功能**: 现代浏览器需要用户交互后才能播放音频
2. **语音权限**: 语音识别需要用户授权麦克风权限
3. **性能考虑**: 3D场景在低端设备上可能影响性能
4. **网络依赖**: CDN资源需要网络连接
